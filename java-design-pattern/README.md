# 通过简单的例子介绍Java23种设计模式
# 一 概述
>本笔记围绕：***由“产品”推导“设计”*** 为核心思想，先展示产品，再引出产品对应的设计模式，通过设计模式介绍产品是如何组装起来的；  
>产品：指如何使用设计模式设计的代码；

# 二 文档分类
>* markdown: 存放设计模式的笔记，笔记包含源码；
>* pattern: 可直接运行的源码，帮助理解笔记；
>   * producer: 设计模式中的生产者；
>   * product: 设计模式中生产者的产出物；
>   * producer + product = 设计模式所有代码，用Test类进行应用；
>* uml：设计模式的UML图汇总；
